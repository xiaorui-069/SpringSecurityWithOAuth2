# SpringSecurityWithOAuth2
This repository contains a demonstration of OAuth2 implementation using Spring Framework. The project is divided into three components:

1. **Authorization Server**: Issues OAuth2 tokens after authenticating users.
2. **Resource Server**: Protects APIs and validates tokens.
3. **OAuth2 Client**: Accesses secured resources using tokens.

## Features

- OAuth2 Authorization Code flow
- JWT token validation
- Simple in-memory user authentication for demonstration purposes
